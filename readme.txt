This project is organized into a single Jupyter notebook that walks through the full workflow from data preparation to analysis. The notebook begins by loading the Yelp dataset and preparing a manageable subset of the data for processing. It then moves through steps such as merging relevant datasets, selecting useful columns, and creating a structured dataset for further use.
In the next section, the notebook focuses on transforming the data and preparing it for modeling. This includes creating additional features and organizing the data into a format suitable for analysis. The workflow is designed to be sequential, so each step builds on the previous one.
The final part of the notebook includes visualizations and analysis, where the processed data is explored to identify patterns and observations. The results are presented through graphs and summary outputs that help illustrate the overall findings.
The notebook is intended to be run from top to bottom, with all required datasets placed in the same directory.
